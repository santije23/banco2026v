# LABORATORIO DEL CURSO DE ARQUITECTURA DE SOFTWARE

## Para iniciar el proyecto
1. Se requiere ingresar en la carpeta banco-frontend para poder iniciar los servicios en **REACT**
`cd banco-frontend `.
2. Instalar **npm** mediante `npm install`.
3. Iniciar el servicio de react mediante `npm start ` el cual inicia en la URL http://localhost:3000/
4. Iniciar servicio en Spring Boot desde la clase `Banco2026vApplication`
